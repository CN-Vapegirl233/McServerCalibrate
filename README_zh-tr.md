一個我的世界Java版伺服器外掛程式，用於校驗玩家的身份和密碼，以保護你的我的世界伺服器。 目前外掛程式可以支援Java版1.20.x~1.21.x。 （只測試了1.20.1、1.20.4、1.21.8，其他版本請自行測試） 運行外掛程式需要你的伺服器是支援安裝外掛程式的外掛程式端（如Spigot、Bukkit、Paper）或混合端（如Mohist、Arclight）。
