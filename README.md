一个我的世界Java版服务器插件，用于校验玩家的身份和密码，以保护你的我的世界服务器。 目前插件可以支持Java版1.20.x~1.21.x。（只测试了1.20.1、1.20.4、1.21.8，其他版本请自行测试） 运行插件需要你的服务器是支持安装插件的插件端（如Spigot、Bukkit、Paper）或混合端（如Mohist、Arclight）。
